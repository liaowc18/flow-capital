# Flow Capital - 功能展示

## 🎨 现代金融风格设计

### 配色方案
- **主色调**: 深蓝色 (#0F2557) - 专业、稳重
- **金色强调**: 金色 (#D4AF37) - 高端、价值感
- **渐变效果**: 深蓝→浅蓝的金融风格渐变
- **玻璃态**: 半透明背景 + 模糊效果

### 视觉特效
- ✨ 玻璃态效果 (glass effect)
- 🎭 拟态设计 (neumorphic)
- 🌈 金融渐变背景
- ✨ 流畅过渡动画
- 💫 徽章脉动效果

## 📱 核心功能页面

### 1. 首页 - AI 智能推荐
**访问路径**: 默认首页

**功能亮点**:
- AI 推荐项目卡片（Tinder 式滑动交互）
- 匹配度进度条（92%匹配）
- 项目标签（行业、阶段、地区）
- 推荐理由说明
- 快速操作（跳过/感兴趣）

**交互效果**:
- 卡片滑出动画（向左滑出带旋转）
- 按钮点击涟漪效果
- 页面渐显动画

### 2. 项目详情页
**访问方式**: 点击任意项目卡片

**展示内容**:
- 项目头部（渐变背景 + 匹配度）
- 融资信息卡片（4个数据指标）
- 项目亮点（带绿色对勾）
- 风险提示（黄色警告框）
- 收藏功能（心形图标）

**设计特点**:
- 金融渐变头部
- 拟态风格数据卡片
- 绿色成功色高亮
- 玻璃态顶部栏

### 3. 筛选面板
**访问方式**: 点击顶部筛选图标

**筛选维度**:
- 🏢 行业领域（9个选项）
- 📊 融资阶段（6个阶段）
- 💰 融资金额（5个区间）
- 📍 项目地区（7个城市）
- 🎯 匹配度滑块（0-100%）
- 📋 排序方式（5种排序）

**交互特点**:
- 从底部滑入动画
- 标签选中状态（深蓝底色）
- 渐变滑块
- 实时条件计数

### 4. 消息中心
**访问方式**: 底部导航 "消息" 或顶部铃铛图标

**消息类型**:
- 系统通知（蓝色图标）
- 项目动态（绿色图标）
- 创始人消息（金色图标）

**设计元素**:
- 红点提醒（脉动效果）
- 圆角头像
- 渐变图标背景
- 卡片悬浮效果

### 5. 个人中心
**访问方式**: 底部导航 "我的"

**核心信息**:
- 用户信息卡片（渐变背景）
- 统计数据（已投/发布/收藏）
- 快捷功能（4宫格）
- 设置入口

**视觉设计**:
- 金融渐变头部
- 玻璃态统计卡片
- 彩色功能图标
- 白色设置列表

### 6. 我的项目
**访问方式**: 底部导航 "我的项目"

**项目管理**:
- 项目状态统计（3个状态）
- 进行中项目列表
- 融资进度展示
- 已完成项目

**数据展示**:
- 拟态统计卡片
- 进度条动画
- 状态标签
- 投资人意向数

### 7. 发布项目
**访问方式**: 底部导航中心 "+" 按钮

**发布流程**:
- 步骤指示器（1/2/3）
- 基本信息表单
- 行业选择器
- 融资阶段按钮组
- 项目简介输入

**表单设计**:
- 圆角输入框
- 深蓝聚焦边框
- 必填项标记
- 金融渐变按钮

## 🎯 交互细节

### 动画效果
1. **页面切换**: 渐显 + 缩放 (0.4s cubic-bezier)
2. **卡片滑出**: 向左旋转滑出 (0.5s)
3. **Toast 提示**: 从底部滑入 (0.4s)
4. **按钮点击**: 缩放 + 涟漪效果
5. **徽章脉动**: 2s 循环脉动
6. **卡片悬浮**: 向上 4px + 阴影增强

### 视觉反馈
- ✅ 按钮点击缩放至 96%
- ✅ 涟漪扩散效果（白色半透明）
- ✅ 悬浮卡片上移
- ✅ Toast 自动消失（2秒）
- ✅ 徽章脉动提醒

## 🎨 配色参考

### 主题色
```css
--primary: #0F2557      /* 深蓝 - 主色调 */
--secondary: #1E40AF    /* 蓝色 - 次要色 */
--accent: #D4AF37       /* 金色 - 强调色 */
--success: #059669      /* 绿色 - 成功 */
--warning: #F59E0B      /* 橙色 - 警告 */
--error: #DC2626        /* 红色 - 错误 */
```

### 渐变组合
```css
/* 金融风格渐变 */
linear-gradient(135deg, #0F2557 0%, #1E40AF 50%, #2563EB 100%)

/* 金色渐变 */
linear-gradient(135deg, #D4AF37 0%, #F59E0B 100%)

/* 卡片渐变 */
linear-gradient(145deg, rgba(255,255,255,0.95) 0%, rgba(255,255,255,0.85) 100%)
```
