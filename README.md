# Flow Capital - 智能融资对接平台

> 现代金融风格的项目融资与投资对接平台

## 项目概览

Flow Capital 是一个基于 AI 智能匹配的融资项目对接平台，采用现代金融风格设计，为创业者和投资人提供高效的项目对接服务。

### ✨ 核心特性

- **🎯 AI 智能匹配** - 基于用户偏好的智能项目推荐系统
- **💼 现代金融风格** - 深蓝色主题 + 金色点缀，专业且优雅
- **📱 移动端优先** - 完美适配移动设备，流畅的手势交互
- **🎨 精致动画** - 玻璃态、拟态设计配合流畅的过渡动画
- **🔍 高级筛选** - 多维度筛选条件，精准定位目标项目
- **💬 消息中心** - 实时项目动态与投资人沟通

### 数据模型
```javascript
{
  id: 项目ID,
  title: 项目标题,
  industry: 行业分类,
  stage: 融资阶段,
  location: 项目地区,
  amount: 融资金额,
  valuation: 投前估值,
  minInvest: 最低投资额,
  interested: 意向人数,
  match: 匹配度（0-100）,
  icon: 项目图标,
  team: 团队介绍,
  highlight: 项目亮点,
  reason: 推荐理由,
  details: 详细信息数组
}
```

## 🎨 设计风格

### 配色方案
- **主色调 Primary**: `#0F2557` (深蓝色) - 专业、稳重
- **次要色 Secondary**: `#1E40AF` (蓝色) - 科技感
- **强调色 Accent**: `#D4AF37` (金色) - 高端、价值感
- **成功色 Success**: `#059669` (绿色) - 积极、正向
- **警告色 Warning**: `#F59E0B` (橙色) - 提醒
- **错误色 Error**: `#DC2626` (红色) - 警示

### 视觉特效
- **渐变背景**: 从深蓝到浅蓝的金融风格渐变
- **玻璃态**: 半透明背景 + 模糊效果
- **拟态设计**: 内外阴影打造立体感
- **金色点缀**: 关键信息使用金色高亮
- **流畅动画**: cubic-bezier 缓动函数

## 📖 开发指南

### 本地开发

```bash
# 安装依赖
npm install

# 构建项目
npm run build

# 启动开发服务器（使用 PM2）
fuser -k 3000/tcp 2>/dev/null || true
pm2 start ecosystem.config.cjs

# 查看服务状态
pm2 list

# 查看日志（非阻塞）
pm2 logs flow-capital --nostream

# 停止服务
pm2 delete flow-capital
```

### 构建和部署

```bash
# 构建生产版本
npm run build

# 预览构建结果
npm run preview

# 部署到 Cloudflare Pages
npm run deploy
```

## 📝 用户使用指南

### 投资人使用流程
1. **浏览项目** - 在首页查看 AI 推荐的项目卡片
2. **匹配评估** - 根据匹配度和推荐理由快速判断
3. **详细了解** - 点击卡片查看项目完整信息
4. **表达意向** - 点击"感兴趣"发送投资意向
5. **筛选定制** - 使用筛选功能精准定位目标项目
6. **跟踪管理** - 在"我的"页面管理投资项目

### 创业者使用流程
1. **发布项目** - 填写项目基本信息和融资需求
2. **完善资料** - 上传 BP、财务报表等材料
3. **等待匹配** - 系统自动匹配合适的投资人
4. **查看意向** - 在"我的项目"中查看投资意向
5. **沟通对接** - 通过消息中心与投资人沟通
6. **进度追踪** - 实时查看融资进度
